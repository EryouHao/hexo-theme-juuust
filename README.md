# hexo-theme-Daily

一款适用于设计师的 hexo 主题。

## 一览
![Juuust](./screenshot.png)

## 使用
1. 在你的 hexo 博客根目录执行如下命令:
    ```git
    $ git clone https://github.com/leadream/hexo-theme-juuust.git themes/juuust
    $ npm install --save hexo-renderer-sass
    ```

2. 把 `config.yml` 中的 `theme` 改成 `juuust`
    ```yml
    # theme: landscape
    theme: juuust
    ```

3. 运行: `hexo s`
